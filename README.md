# OthelloGame
Java Based OthelloGame with Client &amp; Server Interaction.

Change the IP Address in the code.
To Create the Class:

Run:
javac Gamehandler.java OthelloGame.java OthelloGameClientGUI.java OthelloServer.java
To Delete the Class:

Run:
del *.class
To Run the Server:

Run:
java OthelloServer
To Run the Client:

Run:
java OthelloGameClientGUI
